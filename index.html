Automatic Regulation of pH and EC for NFT Hydroponics System

×

Abstract

Hydroponics is an artificial technique of cultivating plants in a liquid solution composing of essential nutrients as a replacement to the use of soil. A hydroponic technique in which the dissolved nutrients are continuously flowing over the roots of plants and recirculating in the nutrient tank is known as the Nutrient Film Technique (NFT). This study aims to determine if having a consistent value within the recommended range for the conductivity level and pH of the hydroponic solution in an NFT hydroponics system will produce better growth for arugula. The researchers recorded the plant parameters for the automatic and manual system every 3 days for 30 days. Comparing the data obtained, plants cultivated on the automatically regulated system showed better plant growth. The plants showed a 39.5 mm higher plant height, 2.5 more number of leaves, 13.625 mm wider leaves, and 25.92 mm longer leaves on the automatically regulated system.

A Braille Learning System for the Blind using ATmega328p

×

Abstract

This design project was developed to address the client's need for a device to assist the blind in their foundation in learning the braille and monitor their progress. There are three modes in the device that will help the users learn and know their progress - Braille Pattern Recognition, Voice Input Braille Recognition, and Guess the Braille Pattern. The device will operate using ATmega328P as the microcontroller. It will also be using six pushbuttons and six push/pull solenoids to mimic the braille pattern that was formed by the user. The device can be controlled using a mobile application and it will be connected to the device via Wi-Fi. The mobile application will be used by the facilitator. The mobile application supports voice recognition which will accept input from the user, and it can also be used to test the knowledge of the user about the Braille. Using the three modes of the device, its functionalities and accuracy were tested. Based on the tests conducted and analysis of the results, it can be concluded that the device can accurately form the corresponding braille pattern from the input of the user.
